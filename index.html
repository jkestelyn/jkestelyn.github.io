Cloudera's Open Source Projects

Welcome to Cloudera's GitHub account, where sourcecode for Apache Hadoop ecosystem projects is managed (in many cases, prior to Apache incubation). The major projects are listed below. All code here carries the Apache License.

See all Repositories

Impala

Cloudera Impala is a modern, distributed SQL query engine for Hadoop. It complements Apache Hive's batch-oriented processing with capability for low-latency workloads.

Repo

Search

Cloudera Search is Apache Solr integrated with CDH, including Apache Lucene, Apache SolrCloud, Apache Flume, MapReduce & HDFS, and Apache Tika.

Repo

Hue

Cloudera Hue is an extensible Web GUI that makes Hadoop easy to use. It includes pre-built apps for Impala, Search, Apache HBase, and more.

Repo

CDK

Cloudera Development Kit (CDK) is a set of libraries, doc, examples, and tools for making it easier to build applications on top of (or integrated with) Hadoop.

Repo

Cloudera Manager API Client

This project contains all the source, examples, and documentation you need to easily build a Cloudera Manager client in Java or Python.

Repo

Llama

Llama is a YARN Application Master that mediates the management and monitoring of cluster resources between Impala and YARN.
