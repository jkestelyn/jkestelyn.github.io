Apache Hadoop To Go!

The Cloudera Development Kit (Apache License), or CDK for short, is a set of libraries, tools, examples, and documentation focused on making it easier to build systems on top of the Hadoop ecosystem.

Codifies expert patterns and practices for building data-oriented systems and applications

Lets developers focus on business logic, not plumbing or infrastructure

Provides smart defaults for platform choices

Supports gradual adoption via loosely-coupled modules

This project is organized into modules. Modules may be independent or have dependencies on other modules within the CDK. When possible, dependencies on external projects are minimized.
